# 🖼️ Image Classification with Neural Networks

A deep learning project that uses a **Convolutional Neural Network (CNN)** built with **TensorFlow** and **Keras** to classify images from the **CIFAR-10** dataset into 10 different categories.

---

## 📌 Features

- Image preprocessing and normalization
- Data visualization with Matplotlib
- CNN model for image classification
- Model training and validation
- Image prediction on test data
- Performance evaluation using accuracy

---

## 🛠️ Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- OpenCV
- Matplotlib

---

## 📂 Dataset

The project uses the **CIFAR-10** dataset, which contains **60,000** 32×32 color images across **10 classes**:

- ✈️ Plane
- 🚗 Car
- 🐦 Bird
- 🐱 Cat
- 🦌 Deer
- 🐶 Dog
- 🐸 Frog
- 🐴 Horse
- 🚢 Ship
- 🚚 Truck

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/aatifamugheer/Image-Classification-with-Neural-Networks
```

2. Navigate to the project folder

```bash
cd Image-Classification-with-Neural-Networks
```

3. Install the required libraries

```bash
pip install tensorflow numpy matplotlib opencv-python
```

---

## ▶️ Run the Project

```bash
python imageclassification.py
```

The dataset will be downloaded automatically on the first run.

The CNN architecture consists of:

- Max Pooling Layers
- Flatten Layer
- Dense Hidden Layer
